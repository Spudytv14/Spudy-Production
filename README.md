# Spudy Library | Premium Learning

A premium, glassmorphism-styled Single Page Application (SPA) for curating and consuming educational content (Books, Music, Videos, Courses).

## Features

- **Premium UI:** Dark mode with glassmorphism, 3D tilt effects, and particle background.
- **Library System:** Filter by category, genre, rating, and sort items.
- **User Accounts:** LocalStorage-based authentication (Login/Register) and Profile management.
- **Persistence:** Saved items, course progress, and user settings are stored locally.
- **Media Integration:**
  - **YouTube:** Auto-fetching of titles and thumbnails.
  - **Persistent Player:** Listen to music while browsing.
- **Responsive:** Fully optimized for desktop and mobile.

## Tech Stack

- **Core:** Vanilla JavaScript, HTML5, CSS3
- **Styling:** Tailwind CSS (CDN)
- **Effects:** Three.js (Particles), Vanilla Tilt, AOS (Scroll Animations)
- **Icons:** FontAwesome

## Setup

Simply open `index.html` in any modern web browser. No build step required.
